# Grow With Me Backend

## Summary

This is the backend for the Grow With Me web app. It is a RESTful API that allows users to create, read, update, and delete users (consultants and farmers), crops, etc.

## Features

- CRUD

## Technologies

- Java 22
- Spring Boot
- JPA
- Lombok
- Validation
- Pluralize
- PostgreSQL
- Dev Tools
